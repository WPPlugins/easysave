Simple plug-in which enables saving post using shortcuts (Ctrl-S or Cmd-S)

== Description ==

easySave is really simple Word Press plugin which enables saving post by pressing classic shortcuts (Ctrl-S on Windowns and Cmd-S on OS X)
We have made it to facilitate your work as much as possible. Yo are probably used to save most of the documents using shortcuts. So why not do it the same way in the Word Press!

== Installation ==

1. Upload `easySave` folder to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress

== Frequently Asked Questions ==

= How can I use the plugin? =

During the post or page editing just press Ctrl + S (Cmd + S on OS X) when you want to save your post.
